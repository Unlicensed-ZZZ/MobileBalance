Если запуск опроса недоступен и при этом нет открытого расширением окна с активным (идущим в этот момент) опросом, то нажмите эту кнопку

Чтобы опросы запускались в указанное время следует разрешить работу браузера в фоновом режиме в его настройках. Этот параметр находится в подразделе системных настроек браузера.
Название параметра для разных браузеров различается: продолжать работу фоновых приложений, когда ... закрыт; разрешить работу в фоне...; не отключать работающие в фоновом режиме сервисы при закрытии...; ...разрешать в фоне выполнение функций Браузера
Если этот параметр в настройках браузера не установлен, то расширение будет корректно работать с опросами по расписанию только если браузер в момент проведения опроса запущен. При запуске браузера он может предпринять попытку выполнить пропущенный опрос (зависит от браузера).

Включите в настройках разрешений браузера получение оповещений (уведомлений) для расширения и откройте настройки повторно

  • При переходе с автономной программы MobileBalance возможно сохранить собранную в ней историю запросов в файл и импортировать её в хранилище расширения. Загрузить инструкцию по подготовке файла истории запросов программы MobileBalance для импорта в расширение.
  • Воздержитесь от загрузки или сохранения истории запросов во время выполнения очередного опроса. Это может вызвать конфликт работы расширения с хранилищем. Дождитесь завершения текущего опроса.
  • Данные расширения (история запросов, общие настройки, учётные данные для опроса, настройки для провайдеров) технически хранятся во внутренних структурах браузера. Если вы решили переустановить расширение, перенести (дублировать) его на другой компьютер или в другой браузер - сохраните эти данные в файлы до удаления расширения. После (пере)установки расширения вы сможете загрузить в него данные из сохранённых файлов и продолжить работу. Объём донастройки расширения при этом будет минимальным. И, что важно, - не будет утрачена история предыдущих запросов.
Внимание, важно!
При удалении расширения все сохраняемые им данные удаляются вместе с расширением без дополнительных уведомлений со стороны браузера. Это действие необратимо и восстановить данные не получится.

Расширение разработано на основе прототипа - автономной программы MobileBalance для Microsoft Windows. Основанная на взаимодействии с Microsoft Internet Explorer, автономная MobileBalance может работать всё с меньшим количеством провайдеров - они перестают поддерживать Internet Explorer на своих сайтах.
Расширение является одной из альтернатив для получения сведений о текущем балансе на компьютерах / ноутбуках. Поскольку оно рассчитано на работу в среде Chromium-браузеров, то по сравнению с автономной программой у него в активе кроссбраузерность и кроссплатформенность. На текущий момент работоспособность проверена в среде MS Windows (Google Chrome, Yandex browser, Microsoft Edge, Opera) и Linux (Chromium на базе Debian).
Коллекция планинов при необходимости может быть дополнена (обращайтесь, попробуем). Кроме того, в расширении воспроизведена возможность дополнения плагинами, разрабатываемыми сообществом. В отличии от автономной MobileBalance, код расширения полностью открыт и оно может быть дополнено новыми возможностями. Плагины расширения очень близки по алгоритму работы к плагинам прототипа. Но они имеют большую свободу в использовании современных языковых конструкций JavaScript, на котором они (как и само расширение) написаны.
